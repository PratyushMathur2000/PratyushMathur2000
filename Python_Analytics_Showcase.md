## 📌 Executive Summary

Welcome to the **TwinPulse Asset Prognosis Dashboard**—a robust, highly interactive Streamlit application engineered to solve complex industrial equipment challenges through data. Developed as a flagship MBA analytics project, this dashboard bridges the gap between sophisticated machine learning prognostics and intuitive business intelligence (BI) interfaces. 

Designed with operational leads, reliability engineers, and C-suite executives in mind, the platform provides real-time fleet health tracking, advanced statistical forecasting, and granular sensor data exploration via digital twins, all without requiring technical expertise from the end user.

---

## ✨ Key Features & Capabilities

### 🏭 1. Fleet Overview & Detailed Asset View
- **Holistic Monitoring:** View the health status, RUL (Remaining Useful Life), and operational metrics of an entire industrial fleet at a glance.
- **Granular Inspection:** Drill down into individual assets to track historical performance and pinpoint localized anomalies.

### 🧠 2. Machine Learning Prognostics & Digital Twin Sandbox
- **Predictive Maintenance:** Under-the-hood machine learning models (`saved_models`) forecast equipment degradation to prevent costly unplanned downtime.
- **Digital Twin Sandbox:** Interact with a virtual representation of physical assets to monitor real-time simulated conditions and structural health.

### 🔄 3. Production Simulator
- **What-If Scenarios:** Interactive sliders and inputs that allow stakeholders to adjust operational variables (e.g., load, speed, temperature) and immediately see the predicted impact on asset longevity and production output.

### 🤖 4. AI Diagnostic Agent
- **Automated Root Cause Analysis:** An integrated AI assistant that analyzes sensor trends and alerts, providing actionable recommendations to maintenance teams.

## 🛠 Technical Architecture

The project follows a modular, maintainable, and highly scalable software architecture:

- **Frontend / UI:** Streamlit (Multi-page Interactive Web Framework)
- **Data Manipulation:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn (Predictive degradation modeling)
- **Visualization:** Plotly, Matplotlib
- **AI Integration:** LLM-based Diagnostic Agent

## 🚀 Installation & Setup

To run this dashboard locally, follow these steps:

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/twinpulse-analytics.git
cd twinpulse-analytics
```

### 2. Create a Virtual Environment
```bash
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Launch the Dashboard
```bash
streamlit run app.py
```

Navigate to `http://localhost:8501` in your browser to interact with the application.
